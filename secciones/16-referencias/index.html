20. Bibliografía

En esta sección se presentan las fuentes bibliográficas y recursos técnicos consultados durante el desarrollo del proyecto MGR. Las referencias se organizan siguiendo el formato de citación APA (7.ª edición).

Google. (2025). Documentación de Google Material Design. Google Design.

https://m3.material.io/

Mozilla Developer Network. (2025). Documentación de tecnologías web HTML, CSS y JavaScript. Mozilla Foundation.

https://developer.mozilla.org/

React Team. (2025). React Documentation: Learn React. Meta Open Source.

https://react.dev/

Node.js Foundation. (2025). Node.js Documentation.

https://nodejs.org/

PostgreSQL Global Development Group. (2025). PostgreSQL Documentation.

https://www.postgresql.org/docs/

Tailwind Labs. (2025). Tailwind CSS Documentation.

https://tailwindcss.com/docs

Pressman, R. S., & Maxim, B. R. (2020). Ingeniería del software: Un enfoque práctico (9.ª ed.). McGraw-Hill Education.

Schwaber, K., & Sutherland, J. (2020). La Guía Scrum: La guía definitiva de Scrum.

https://scrumguides.org/

Figma. (2025). Figma Design Documentation.

https://help.figma.com/

Oracle. (2025). Java Documentation.

https://docs.oracle.com/

Conclusión

Las fuentes consultadas permitieron fundamentar las decisiones técnicas, metodológicas y de diseño utilizadas durante la construcción del sistema MGR, apoyando el desarrollo del software bajo buenas prácticas de ingeniería y documentación.

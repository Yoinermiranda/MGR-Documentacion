Capítulo 05 · Justificación y alcance

Alcance y Justificación

Alcance

El sistema de gestión de punto de venta permitirán automatizar procesos como la gestión de pedidos, control de mesas, facturación y administración básica del restaurante. Estará dirigido a tres tipos de usuarios: administrador, cajero y mesero, cada uno con funciones específicas dentro del sistema. Entre sus principales características se incluyen la creación de pedidos, visualización del estado de mesas, procesamiento de pagos y generación de reportes. El sistema se desarrollará como una aplicación web utilizando tecnologías modernas, garantizando su funcionamiento en diferentes dispositivos. Sin embargo, no se contemplan en esta etapa funcionalidades como integración con sistemas de pago externos o facturación electrónica oficial, las cuales podrán implementarse en futuras versiones.

Justificación

Este proyecto se justifica por la necesidad de mejorar la eficiencia en la gestión de restaurantes, reduciendo errores y optimizando procesos como pedidos y facturación. Además, permite aplicar conocimientos de desarrollo de software en una solución real, utilizando tecnologías modernas y accesibles para pequeños y medianos negocios.

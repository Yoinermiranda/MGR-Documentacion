Tecnologías utilizadas - Sistema MGR

El sistema POS se desarrolla con tecnologías web modernas. En el frontend se usa React junto con Tailwind CSS para crear interfaces dinámicas y responsivas. En el backend se emplea Node.js con Express para construir una API REST, incorporando JWT para la autenticación. Además, se utiliza Prisma como ORM para gestionar una base de datos relacional. Estas herramientas permiten crear un sistema eficiente, escalable y adaptable a las necesidades del negocio.
